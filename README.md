# Salary Prediction

A simple machine learning project to predict salary based on input features.

## Files

- `train_model.py` - trains the model
- `app.py` - basic app entry point
- `app_ui.py` - user interface script

## Run

1. Train the model:
   ```bash
   python train_model.py
   ```
2. Start the app:
   ```bash
   python app.py
   ```
3. Start the UI (Streamlit):
   ```bash
   streamlit run app_ui.py
   ```
